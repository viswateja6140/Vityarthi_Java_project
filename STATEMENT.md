## Problem Statement
Users often find it difficult to track which movies and TV shows they have watched, which ones they plan to watch later, and their progress in long-running shows.  
This project provides a simple system to manage movies and shows along with a watchlist and progress tracking functionality.

---

## Scope of the Project

### Included in Scope
- Adding movies and TV shows  
- Tracking watched status for movies  
- Tracking progress (season, episode) for shows  
- Watchlist functionality for both movies and shows  
- Searching by title or genre  
- Updating movie and show details  
- Saving and loading data from CSV files  

### Excluded from Scope
- User authentication or user profiles  
- Recommendation algorithms  
- Online database or cloud integration  
- GUI front-end (currently console-based)

---

## Target Users
- Students learning Java and OOP concepts  
- Movie and TV show enthusiasts who want a personal catalog  
- Beginners learning file handling and data persistence  

---

## High-Level Features
- Movie Management (add, update, search, watchlist)  
- Show Management (add, update progress, search, watchlist)  
- Display watchlist  
- CSV-based storage  
- Modular object-oriented structure using multiple packages and classes
